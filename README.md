# 矿物纪元 Mineral Evolution

这是一个关于锰矿物表生环境进化与污染物去除的网页版科研小游戏。

## 功能

- 自由进化：调节 pH、温度、氧气、Mn²⁺、离子、有机酸和反应时间。
- 关卡模式：按照目标条件生成指定矿物。
- 污染物挑战：苯酚、四环素、诺氟沙星、甲醛和甲苯。
- 矿物图鉴：自动记录已解锁矿物。
- 实验记录：保存、删除和导出 JSON。
- PWA：支持添加到 iPhone 主屏幕。

## 在线访问

GitHub Pages 启用后，访问地址通常为：

```text
https://tcj1234.github.io/Mineral-Evolution/
```

## iPhone 添加到桌面

1. 用 Safari 打开在线地址。
2. 点击分享按钮。
3. 选择“添加到主屏幕”。
4. 桌面会出现“矿物纪元”图标。

## 说明

这是游戏化规则模型，不是真实热力学或动力学预测软件。适合科研科普、原型展示和实验思路整理。
